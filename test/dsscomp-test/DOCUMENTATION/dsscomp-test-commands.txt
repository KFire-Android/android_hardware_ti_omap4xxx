Note 1:
-------
For TV out test cases: use an additional parameter of disp_id=1
while executing the test case wherever applicable
(or)
modify the DEFAULT_DISPLAY to TV_OUT in dsscomp_test.h before
compiling the test case

Note 2:
-------
Please make sure that all the test images are available in the same folder as test binary
By default the following test images are used for various test cases:

TEST_IMAGES/640X480/butterfly.rgb32 	- ARGB32 image 640 * 480
TEST_IMAGES/640X480/xena.uyvy		- uyvy image 640 * 480
TEST_IMAGES/640X480/xena.yuv		- NV12 image 640 * 480

(all available in dsscomp-test/TEST_IMAGES directory)
These test cases uses single frame of the NV12 images,
even though there are multiple frames in the NV12 images
(with ".yuv" extension) at dsscomp-test/TEST_IMAGES
